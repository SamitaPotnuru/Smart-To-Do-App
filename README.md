# Smart To-Do App

A modern **To-Do List Web Application** built using **HTML, CSS, and JavaScript** to help users manage daily tasks efficiently.

---

## 📌 Features

- ➕ Add new tasks
- ✏ Edit existing tasks
- ❌ Delete tasks
- ✔ Mark tasks as completed
- 🔎 Search tasks
- 📅 Add due dates
- ⚡ Task priority (High / Medium / Low)
- 📂 Filter tasks (All / Completed / Pending)
- 💾 Save tasks using Local Storage

---

## 🔮 Future Improvements

The project will be upgraded with the following features:

- 👤 User **Login / Signup Authentication**
- 🗄 Database integration (MongoDB / Firebase)
- 🔄 Load **previous task history after login**
- 🌐 Backend API integration
- ☁ Cloud deployment

Example workflow:

```
User Login
   ↓
Get User ID
   ↓
Fetch tasks from database
   ↓
Display previous task history
```
